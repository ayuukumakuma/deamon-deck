# Phase 16: ホーム画面への動線

## 概要
ホーム画面（WelcomePanel）へ戻る動線を2つ設置する。サイドバー上部のタイトル/ロゴクリックと、選択中サービスの再クリックによる選択解除。

## 前提
- Phase 15 完了

## タスク

### サイドバー上部のホームボタン
- [x] `src/components/layout/Sidebar.tsx` を修正
  - [x] 「Services」タイトル部分（またはロゴ/アイコン）をクリック可能にする
  - [x] クリック時に `setSelectedServiceLabel(null)` を呼び出し
  - [x] カーソルを `pointer` に変更
  - [x] ホバー時の視覚フィードバック（opacity変化やアンダーライン等、控えめなスタイル）
  - [x] サービス未選択時はクリック不可またはスタイルを抑制

### サービス再クリックによる選択解除
- [x] `src/components/service-list/ServiceRow.tsx` を修正
  - [x] 選択中のサービス行をクリックした場合、選択を解除する（トグル動作）
  - [x] `selectedServiceLabel === label` の場合は `setSelectedServiceLabel(null)` を呼出
  - [x] それ以外は従来通り `setSelectedServiceLabel(label)` を呼出

### i18n（必要な場合）
- [x] `src/lib/i18n.ts` にツールチップ等の翻訳キーを追加
  - [x] `sidebar.homeTooltip` — ホームボタンのツールチップ（例:「ホームに戻る」）

## 完了条件
- サイドバーのタイトル部分をクリックするとホーム画面（WelcomePanel）に遷移する
- 選択中のサービスを再クリックすると選択が解除されホーム画面に戻る
- ホーム画面表示中にタイトル部分のクリックが無駄に動作しない
- 既存のサービス選択・ナビゲーション機能が正常動作する
